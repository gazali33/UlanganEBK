Soal Online Estimasi Biaya Konstruksi
Aplikasi web untuk ujian online mata pelajaran Estimasi Biaya Konstruksi untuk kelas XI DPIB SMK Muhammadiyah Berau.

Fitur
Sistem token akses untuk keamanan
30 soal acak dari bank 40 soal
Timer 25 menit
Navigasi soal interaktif
Penyimpanan hasil ke Google Sheets
Responsive design
Teknologi
HTML5
CSS3 dengan Bootstrap 5
Vanilla JavaScript
Google Apps Script untuk backend
Cara Menggunakan
Masukkan token akses yang valid
Pilih nama siswa dari daftar
Kerjakan soal dalam waktu 25 menit
Lihat hasil dan share jika diperlukan
Token Valid
EBK2023
KONSTRUKSI
XI2023
DPIB123
Integrasi Google Sheets
Untuk menyimpan hasil ke Google Sheets:

Buat Google Sheet baru
Buat Apps Script dengan kode yang disediakan
Deploy sebagai Web App
Ganti URL_WEB_APP di file JavaScript dengan URL dari Apps Script
